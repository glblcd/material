## Loops
```python
names = ["John", "Paul", "George", "Ringo"]
for beatle in names:
    print (beatle)
```
    John
    Paul
    George
    Ringo

## Loops
Also tuples:
```python

for item in hostport:
    print (item)
```

## Loops
* nested *control flow statements*
```python
names = ["John", "Paul", "George", "Ringo"]
for beatle in names:
    if len(beatle) > 4:
        print (beatle)
```
    George Ringo

## Loops
`while` loops until condition returns false
```python
num = 0
while num < 5:
    num = num + 1 
    print (num)
```
    1 2 3 4 5

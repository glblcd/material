## String
Use "" or '', doesn't matter:
```sh
>>> print ("hello" == 'hello')
True
```

+++
## String
...or Unicode (with care!)
```sh
>>> command = "⌘"
>>> command
'\xe2\x8c\x98'
>>> print (command)
⌘
```

+++
